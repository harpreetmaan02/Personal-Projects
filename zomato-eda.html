Zomato EDA Project

Project Overview

In this exploratory data analysis (EDA) project focused on Zomato data, we aimed to uncover valuable insights and trends within a vast dataset of restaurants. The analysis involved studying various parameters such as restaurant locations, cuisines, user ratings, and more to provide a comprehensive view of the Zomato ecosystem.
